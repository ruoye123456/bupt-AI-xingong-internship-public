# 2026年1月北邮专业实习文件
## 写在前面

本项目基于北京邮电大学人工智能学院信息工程专业2026年1月专业实习文件整理而成，旨在为后来人提供参考和帮助。感谢所有为本项目做出贡献的同学。

## 关于专业实习实验部分

使用Huwei的eNSP平台进行实验，具体实验内容和步骤有文档，2026
年只做了实验手册里的2和3两个部分，照着手册做就行。（后面如有变化，就请列位诸公自力更生）。

- .efz文件可以使用Bandzip解压，里面有.cfg

## 小巧思

- kimi的PPT生成很好用，本项目的PPT由kimi生成，做出略微修改后即可使用。
- 大部分内容都可以使用GPT5.1和Gemini 3生成。
- 如果你在华玉大厦，出门左转路口的商场里-1层米村拌饭很好吃。
